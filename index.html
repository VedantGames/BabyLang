Bhailang is dynamically typed toy programming language written in Javascript.
General

hey baby chalo shuru karte h is the entrypoint for the program and all program must end with bas baby ab ho gya Anything outside of it will be ignored.

This will be egnored

hey baby chalo shuru karte h
   // Write code here
bas baby ab ho gya

Variables

Variables are declared using baby ye h .

hey baby chalo shuru karte h
    baby ye h a = 10
    baby ye h b = "heyy"
    baby ye h c = 25
    a = a + 2
    b = 50
    c = c * a
bas baby ab ho gya
Types

Numbers and strings are like other languages. Null values can be denoted using nalla . sahi and galat are the boolean values.

hey baby chalo shuru karte h
    baby ye h a = 2
    baby ye h b = 2 + 2
    baby ye h c = "I Love You"
    baby ye h d = sahi
    baby ye h e = galat
    baby ye h f = nalla
    baby ye h g = c
bas baby ab ho gya
Output

Use baby bolo to print anything. It can automaticaly detect strings and variables. No need to use "" to print strings.

hey baby chalo shuru karte h
    baby ye h a = 5
    baby ye h b = 10

    baby bolo a ki value h, a
    baby bolo b ki value h, b
    baby bolo dono ka sum h, a + b
bas baby ab ho gya

Conditions

BabyLang supports if-else-if ladder construct , agar baby block will execute if condition is sahi , otherwise one of the subsequently added nahi to baby blocks will execute if their respective condition is sahi , and the warna block will eventually execute if all of the above conditions are galat .

chalo baby shuru karte h
    baby ye h a = 2
    baby ye h b = 4

    baby agar a > b {
       baby bolo a bada h by, a - b
    }
    nahi to baby a < b {
       baby bolo b bada h by, b - a
    }
    warna {
       baby bolo dono barabar h
    } bas baby ab ho gya

Loops

Statements inside jab tak baby blocks are executed as long as a specified condition evaluates to sahi . If the condition becomes galat , statement within the loop stops executing and control passes to the statement following the loop. Use bas karo baby to break the loop and agla dekho baby to continue within loop.

hey baby chalo shuru karte h
    baby ye h a = 0

    baby jab tak a < 5 {
       baby bolo a 5 se chhota b by, 5 - a
       a = a + 1
    }

bas baby ab ho gya
